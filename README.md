# The Ambler Institute for Ethical AI

Static website for The Ambler Institute for Ethical AI, a UK Community Interest Company (incorporating).

Deployed via GitHub Pages at [theamblerinstitute.org](https://theamblerinstitute.org).

## Setup

1. Enable GitHub Pages in repository Settings > Pages, deploying from the `main` branch root.
2. Configure DNS for `theamblerinstitute.org` to point to GitHub Pages (see [GitHub docs](https://docs.github.com/en/pages/configuring-a-custom-domain-for-github-pages)).
3. Replace the Formspree placeholder endpoint in `index.html` with a live form backend (see comments in the HTML).

## Files

- `index.html` - Single-page website (self-contained with inline CSS)
- `favicon.svg` - SVG favicon
- `apple-touch-icon.png` - 180x180 touch icon for iOS
- `CNAME` - Custom domain configuration for GitHub Pages
